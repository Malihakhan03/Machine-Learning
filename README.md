# Machine Learning Lab Practicals

This repository contains the implementation of Machine Learning laboratory practicals using Python. The practicals are executed using Google Colab and cover fundamental machine learning algorithms and techniques used for data analysis, classification, regression, and clustering.

## Tools and Libraries
- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- TensorFlow / Keras

## List of Practicals

1. **Pre-Lab:** Understanding Google Colab and Python libraries for Machine Learning (NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn)

2. **Linear Regression**  
   Implement Linear Regression for predicting continuous target variables and evaluate model performance using appropriate metrics.

3. **Logistic Regression**  
   Implement Logistic Regression for binary or multiclass classification and analyze classification performance.

4. **Decision Tree**  
   Build and evaluate a Decision Tree model for classification tasks and study the effect of tree depth and splitting criteria.

5. **Random Forest**  
   Implement the Random Forest algorithm and evaluate model performance using suitable evaluation metrics.

6. **Artificial Neural Network (ANN)**  
   Design and implement an Artificial Neural Network for classification problems and analyze its learning performance.

7. **Support Vector Machine (SVM)**  
   Apply SVM for classification and analyze the effect of different kernel functions on model accuracy.

8. **Principal Component Analysis (PCA)**  
   Apply PCA for dimensionality reduction and visualize the transformed feature space.

9. **K-Means Clustering**  
   Implement K-Means clustering algorithm and analyze clustering results using evaluation techniques.

10. **Hierarchical Clustering**  
    Perform Hierarchical Clustering using agglomerative methods and interpret dendrograms for cluster formation.

11. **Machine Learning Model Deployment**  
    Deploy a trained machine learning model using a suitable platform and test it with real-time or unseen data.

12. **Post-Lab: Complete Regression Problem**  
    Implement a complete regression pipeline including data preprocessing, model building, evaluation, and result interpretation.
